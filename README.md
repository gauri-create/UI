# 🌱 The Life of Pea — An Unlikely Developer

An interactive storytelling web project that represents a developer’s journey — from beginner to experienced engineer — using animations, humor, and UI effects.

---

## 🚀 Live Demo

[View Project](https://developersjourney.netlify.app/))

---

## 📌 Features

* Fully interactive single-page experience
* Animated character (Pea) with multiple states
* Smooth scrolling storytelling
* GSAP-powered animations
* Custom cursor and UI effects
* Responsive design

---

## 🛠️ Technologies Used

* HTML5
* CSS3 (inside HTML)
* JavaScript (inside HTML)
* GSAP Animation Library

## 🌐 Deployment

Deployed using Netlify.

Steps:

* Upload project or connect GitHub repo
* Ensure `index.html` is in root
* Deploy with default settings

---

## ⚠️ Common Issues

* Site shows 404 → `index.html` not in root
* CSS/JS not working → wrong paths
* Blank page → deployment misconfigured
